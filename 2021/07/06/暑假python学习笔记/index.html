暑假python学习笔记

发表于 2021-07-06 更新于 2021-07-08

“长风破浪会有时，直挂云帆济沧海。”

暑期学习python的目的：

  1. 掌握深度学习常用框架；
  2. 可自行搭建图像相关算法，实现图像分类，人脸识别；
  3. 可自行搭建NLP常见算法，实现文本分类、问答系统等；
  4. 掌握GAN网络相关算法，实现图像自动上色、图像美化等。

简介：Python意为大蟒蛇，诞生于1989年。目前更新到3.0版本。是一种跨平台的计算机编程语言，是人与计算机交流的工具，开发环节中没有编译这个环节，是一种交互式语言，可以在>>>后面直接写代码，是一门面向对象的编程语言。

搭建开发环境：√

基础语法：

1.输出函数print()函数

2.转义字符

\n 表示 换行

\t 4格是一个制表位，占满了重新开

\r 覆盖之前的内容,光标移动到本行的开头

\b 退一个格

\\ 四个杠

print(‘老师说：'大家好'‘)

print(r’hello\nworld’) #原字符，不希望转义字符起作用。

主要事项：最后一个字符不能是 \

3.二进制与字符编码

计算机只识别二进制

print(ord(‘王’)) 输出：29579

print(chr(29579)) 输出：王

4.Python中的标识符和保留字

查看保留字：

import keyword

print(keyword.kwlist)

输出结果如下：[‘False’, ‘None’, ‘True’, ‘and’, ‘as’, ‘assert’, ‘async’, ‘await’, ‘break’, ‘class’, ‘continue’, ‘def’, ‘del’, ‘elif’, ‘else’, ‘except’, ‘finally’, ‘for’, ‘from’, ‘global’, ‘if’, ‘import’, ‘in’, ‘is’, ‘lambda’, ‘nonlocal’, ‘not’, ‘or’, ‘pass’, ‘raise’, ‘return’, ‘try’, ‘while’, ‘with’, ‘yield’]

所有要起名字的就是标识符，规则：

  • 可以由字母、数字、下划线组成
  • 不能以数字开头
  • 不能是保留字
  • 严格区分大小写
